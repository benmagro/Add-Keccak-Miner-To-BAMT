Add-Keccak-Miner-To-BAMT
========================

This is script will install a keccak algorithm miner within BAMT 1.6.0

#Minable Coins

- CryptoMETH (METH)
- Maxcoin (MAX)
- 365coin (365)

#Instructions

```bash
git clone https://github.com/benmagro/Add-Keccak-Miner-To-BAMT.git
cd Add-Keccak-Miner-To-BAMT/
./keccak.sh
```
